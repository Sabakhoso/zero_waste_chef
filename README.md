Project Overview:
This web application is designed to help users reduce food waste by generating creative recipes based on available ingredients using AI.  

Live Link:
https://sabakhoso-zero-waste-chef.hf.space

Key Features:

AI-Powered Recipes: Uses the Groq API to turn simple ingredients into full meal plans.  

Fast Deployment: Built with FastAPI and hosted on Hugging Face Spaces.  

Eco-Friendly: Specifically focuses on "zero-waste" cooking to help the environment.  

How to Use:

Open the link above.  

Input your ingredients (e.g., tomato, onion, cheese).  

Click Generate Plan to receive your AI-crafted recipe.
