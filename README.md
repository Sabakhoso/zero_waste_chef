# 🥗 Zero-Waste AI Chef
 
**Project Focus:** Artificial Intelligence & Sustainable Solutions  

---

## 📌 Project Overview
The **Zero-Waste AI Chef** is an intelligent assistant designed to help users reduce food waste. By using advanced AI, it takes whatever random ingredients you have left in your fridge and turns them into professional, creative recipes. 

Our goal was to move beyond simple search engines and create a tool that "thinks" about sustainability, providing a specific **Waste Reduction Tip** with every meal generated.

---

## 🛠️ How It Works (Methodology)
We built this project using a "Three-Layer" approach to ensure it is fast, smart, and reliable:

*   **The Brain (AI):** We used the **Groq Llama-3** model to handle the natural language processing. It doesn't just list ingredients; it synthesizes full cooking instructions.
*   **The Server (Backend):** A **FastAPI** server acts as the bridge, sending your ingredients to the AI and bringing back the recipe safely.
*   **The Memory (Storage):** Instead of a slow database, we used **LocalStorage**. This means the app "remembers" your saved recipes and favorites directly in your browser.

---

## ✨ Key Features
*   **AI Recipe Generation:** Turns unstructured text (like "old carrots and a lemon") into a structured recipe.
*   **Smart Archive:** Save your favorite recipes to a digital cookbook with one click.
*   **Recycle Bin:** A "Soft-Delete" feature that lets you move recipes to a trash folder and restore them later if you change your mind.
*   **Sage Chat:** A built-in AI chatbot for quick kitchen advice or cooking tips.
